Welcome to the Jasper documentation

Click on the guides below to learn how to build your own Jasper.

Get the hardware

Jasper runs on Raspbery Pi and a handful of inexpensive, off-the-shelf components.

Download the software

We've figured out the hard stuff. Just download the disk image and plug in your Raspberry Pi.

Write your apps

Jasper provides a dead-simple interface for developers to write their own modules.
